## Project Structure

```
annas-dev-notes/
├── src/
│   ├── content/
│   │   ├── post/         # Blog posts (Markdown / MDX)
│   │   └── note/         # Shorter notes
│   ├── components/       # Astro/UI components
│   ├── layouts/          # Page layouts
│   ├── pages/            # Route pages
│   └── styles/           # Global and theme overrides
├── public/               # Static assets
├── astro.config.mjs
├── tailwind.config.cjs
└── package.json
```
